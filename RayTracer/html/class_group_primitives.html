Notes:

Constructor store the STL vector of pointers to primitives. Intersect just loops through all the primitives in the list and call the intersect routine. Compare thit and return that of the nearest one (because we want the first hit).

Also, the in->primitive should be set to the pointer to that primitive. When you implement acceleration structure, it will replace this class.
